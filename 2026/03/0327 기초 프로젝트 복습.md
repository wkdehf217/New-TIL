# 3/27 기초 프로젝트 복습 

### 프로젝트 기본 뼈대 복습

- 기본적으로 Controller에서 GetMapping, PostMapping 을 통해 필요 파라미터를 전달받고 Service 단으로 넘김
- 전달받는 파라미터는 Dto를 전달받아 Post가 될 수도 있고 UUID 하나만 전달받아서 Get이 될 수도 있음
- Dto는 내부에 String, Integer와 같은 변수를 선언해놓고 그걸 통해 Builder 패턴을 사용하여 Dto를 만듬
- Service 에서는 받아온 데이터로 Repository에 저장 후 리턴
- 사용자에게는 GetMapping 이나 PostMapping을 완료했다면 ResponseEntity.ok 를 통해 결과 리턴
  
  - 한칸 더 나가자면, Service에서 저장 한 이후에 RabbitMQ 나 Kafka 를 통해 메시지를 전달 할 수 있음
